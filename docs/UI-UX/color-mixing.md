# Color Mixing

## 1. Color Theory

1. Có 3 màu gốc là: đỏ, vàng, xanh dương
2. Màu thứ cấp là:

    - Đỏ + Vàng -> Cam
    - Vàng + Xanh -> Xanh lá
    - Xanh + Đỏ -> Tím

3. Màu bổ túc: là màu đối diện trên vòng màu, dùng để tạo độ tương phản mạnh

    - Đỏ - Xanh lá
    - Xanh dương - Cam
    - Vàng - Tím

## 2. Bốn cách phối màu phổ biến trong design

1. Monochrome (1 màu chính)

    - Dùng 1 màu + đậm/nhạt khác nhau
    - VD: xanh đậm - xanh vừa - xanh nhạt.
    - Đặc điểm: an toàn, đẹp, rất hợp UI Dashboard

2. Analogous (màu liền kề)

    - VD: xanh dương - xanh lá - xanh ngọc
    - Đặc điểm: nhìn rất "êm", hợp thiết kế nhẹ nhàng

3. Complementary (đối lập)

    - VD: xanh dương + cam
    - Nổi bật mạnh (dùng cho button, CTA)

4. Triadic (tam giác)

    - VD: đỏ - vàng - xanh
    - Đặc điểm: nhiều màu nhưng vẫn cân bằng

## 3. Nguyên tắc quan trọng

1. Sai phổ biến

    - Dùng quá nhiều màu -> rối
    - Màu quá chói -> nhìn mệt
    - Không có màu chính -> không có trọng tâm

2. Quy tắc 60 - 30 - 10

    - 60%: màu nền chính
    - 30%: màu phụ
    - 10%: màu nhấn
    - VD: nền xám (#979797), text (đen hoặc trắng), accent (xanh hoặc cam)
